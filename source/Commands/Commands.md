# Commands: Discord Slash Command Applications

Commands are code executables inside the discord chat interface.
Using a command triggers interactionCreate event which handles the necessary checks before the command launches.

## Categories
Commands are separated in categories based on permission level required and type of scope the effect of the command has.

- Administrator: Administrative commands such as system setups, guild specific bot settings and higher priviledge management

- Info: Member level commands that provide informations about the server and other public features  

- Social: Member level commands that manage social aspects between members
