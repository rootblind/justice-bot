# objects
JSON and javascript objects used by different systems

- color-defaults: Cosmetic roles
- custom-presence-presets: Custom status for the bot to pick from when setting its presence
- default-presence-presets: A set of presence options for the bot to pick from
- presence-config: Based on this configuration, the bot decides which presence preset to use, delay between auto reassigning it and whether to do it automatically or not
- select_role_options: Contains an array of SelectMenuOptions and dictionaries to convert id to string and string to ad for rank roles
- trigger_words: Regex patters to be used in flagging toxic messages
